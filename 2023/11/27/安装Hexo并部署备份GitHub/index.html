部署博客以及启动博客需要先生成博客文件，具体命令使用方法请看Hexo官方文档

命令 描述
hexo clean 清理静态文件
hexo config 配置文件
hexo deploy 部署博客
hexo server 启动服务
hexo generate 生成静态文件
